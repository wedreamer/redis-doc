刷新 Lua 脚本缓存。

默认情况下，`SCRIPT FLUSH`将同步刷新缓存。
从 Redis 6.2 开始，设置**懒惰自由-懒惰-用户-齐平**配置指令更改为“yes”会将默认刷新模式更改为异步模式。

可以使用以下修饰符之一显式指示刷新模式：

*   `ASYNC`：异步刷新缓存
*   `!SYNC`：同步刷新缓存
